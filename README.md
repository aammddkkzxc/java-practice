### 느낀점
- 마법처럼 느껴지는 동작들이 결국 모두 자바 코드 동작한다는 것
- 개발자의 실수를 줄이고 안정성을 보장하기 위한 제약과 원칙이 중요하다
- 이해도를 높이고 유연하며 편리한 구조를 위한 기술, 디자인 패턴, 그리고 철학이 반영됨을 깨달음
- 기존 방식의 한계를 극복하고 문제를 해결하는 자바의 발전 과정을 알 수 있었다
